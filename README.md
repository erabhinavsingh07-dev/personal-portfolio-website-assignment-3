# personal-portfolio-website-assignment-3

Enhancing with Modern CSS Features1. Use Flexbox and Grid: Implement Flexbox for layout alignment in sections like skills or projects. Use Grid for more complex layouts like project showcases.
2. Responsive Design: Utilize media queries to ensure the website looks good on desktop, tablet, and mobile. Use relative units like %, vw, vh, or em for flexible scaling.
3. Animations and Effects: Add hover effects for interactivity and simple animations like fade-in for visual appeal.

Maintaining Structure- Organize the website into clear sections:
    - Header: Navigation.
    - Hero: Introduction.
    - About: Brief description.
    - Skills: List of skills.
    - Projects: Showcased projects.
    - Contact: Contact form or info.
    - Footer: Copyright or credits.
